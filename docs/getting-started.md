
# Getting Started with MedMaster

This guide will help you get started with the MedMaster platform.

## Account Creation

To get started with MedMaster, you need to create an account:

1. Navigate to the [Sign Up](/signup) page
2. Enter your email address and create a password
3. Complete your profile information
4. Verify your email address

## Exploring the Platform

Once logged in, you can access:

- **Dashboard**: Your personal learning hub
- **Courses**: Browse and enroll in medical courses
- **Webinars**: Join live and recorded webinar sessions
- **Profile**: Manage your account and track your progress

## Free Trial

New users can start with a 7-day free trial that includes:

- Access to selected courses
- Ability to participate in webinars
- Full access to learning tools and features

## System Requirements

MedMaster is a web-based platform that works on:

- Modern web browsers (Chrome, Firefox, Safari, Edge)
- Desktop and mobile devices
- Minimum internet speed of 5 Mbps for video streaming
