
# MedMaster App

## Documentation

This repository includes comprehensive documentation for the MedMaster platform. The documentation is designed to be compatible with GitBook for easy publishing and viewing.

### Documentation Structure

The documentation is located in the `docs/` directory and follows GitBook's recommended structure:

- `README.md` - Introduction and overview
- `SUMMARY.md` - Table of contents for GitBook
- Topic-specific markdown files

### Viewing the Documentation

#### Online (GitBook)

The documentation can be viewed online at [gitbook.com/medmaster](https://gitbook.com/medmaster) (replace with your actual GitBook URL once published).

#### Locally

To view the documentation locally:

1. Install GitBook CLI:
   ```
   npm install -g gitbook-cli
   ```

2. Navigate to the docs directory:
   ```
   cd docs
   ```

3. Serve the documentation:
   ```
   gitbook serve
   ```

4. Open your browser to `http://localhost:4000`

### Updating the Documentation

1. Edit the markdown files in the `docs/` directory
2. Follow GitBook's markdown syntax
3. Update the `SUMMARY.md` file if adding new pages
4. Commit your changes and push to the repository

## About MedMaster

MedMaster is a comprehensive medical education platform designed for medical students, professionals, and educators. The platform offers a wide range of courses, lessons, and resources to help users advance their medical knowledge and skills.
